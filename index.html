tercol DEPRECATED


TerCol is a small, pure Python library that allows you to color and style text output. TerCol has no dependencies and only requires Python 3.7 or newer. It also supports true color. DEPRECATED

Installation DEPRECATED

To install TerCol, you can use pip to install the tercol module. DEPRECATED
